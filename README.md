Portfólio Java - Jefferson Cardoso

Este repositório mostra a evolução dos meus estudos em Java, do básico à Programação Orientada a Objetos (POO).

Estrutura do repositório

# Exercícios Básicos
- AlunoBasico.java: cálculo de média com if/else
- Desconto.java: aplicação de desconto com if
- ServicosSwitch.java: switch case de serviços

# Projetinhos POO
- AlunoPOO: classe `Aluno` com atributos privados, construtor, métodos `calcularMedia` e `verificarSituacao`
- Main.java: execução do programa usando a classe `Aluno`

# Objetivo
Registrar a evolução do aprendizado em Java e praticar conceitos fundamentais de POO.
